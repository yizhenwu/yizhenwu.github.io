如何比较两个double(float)是否相等?

#原理
计算机内部表示小数(float,double型小数)都是有误差的(因为计算机只有0,1)。

#方法

  1. 判断两个小数是否相等,只能判断它们之差的绝对值是不是在一个很小的范围内(我觉得超过其要求的精度即可)。
  2. 如果两个数之差很小,那么就可以认为他们是相等的。

#代码
` //以double为例 bool equal(double num1,double num2) { if(num1 - num2 > -0.000001 && num1-num2 < 0.00001) return true; else return false; }

  2. 注意防止DFS在一次搜索中重复搜索已经搜索过的节点,一般增加book[i]=1,来标识该节点被搜索过了。

快速排序

#算法要点

  1. 基准数;
  2. 哨兵i=0(指向序列最左边);
  3. 哨兵j=length-1(指向序列最右边);
  4. 哨兵j先行,j–,直到发现第一个小于基准数的位置;
  5. 随后,哨兵i后行,i++,直到发现第一个大于基准数的位置;
  6. 交换哨兵i,哨兵j所指向的元素;
  7. goto 3 until 哨兵i位置=哨兵j位置,将当前哨兵i和j所指的元素与基准数交换;
  8. 处理()

#算法实现

#复杂度分析
每次执行都一分为二的递归实现,因此完成排序复杂度O(NlogN)。
